saganIZER

Carl Sagan. Everywhere.

saganIZER is a browser extension with a single purpose: apply a visual theme with Carl Sagan references to the web pages you visit.

GitHub Repository Privacy Policy

Single Purpose

Replace visual elements and some text terms on the open page to create a thematic experience. The extension works on the content displayed to the user and does not offer parallel functionality.

How It Works

Content Script

Runs in the context of the page to adjust the DOM locally in the user's browser.

No Backend

The project has no API or backend server to collect or store usage data.

Wide Host Scope

Necessary to apply the effect consistently across visited websites.
