# saganIZER

Carl Sagan. Everywhere.

`saganIZER` is a browser extension created for [Carl Sagan Day 2014](http://carlsaganday.com/).

## What It Does

- Replaces images on visited pages with random Carl Sagan images.
- Replaces occurrences of the word "God" with "Sagan" (case-insensitive).

## What It Does Not Do

- Make coffee.
- Create a vortex.
- Produce pseudoscience.
