Politica de Privacidade - saganIZER

Esta politica descreve como a extensao saganIZER lida com dados do usuario. O objetivo da extensao e aplicar um efeito visual tematico em paginas visitadas.

Resumo

  • Nao coletamos dados pessoais.
  • Nao vendemos, alugamos ou compartilhamos dados de usuario com terceiros.
  • Nao usamos dados para publicidade, perfilamento, credito ou emprestimo.
  • Nao temos backend proprio para receber dados do usuario.

Dados coletados

A extensao nao coleta nem armazena informacoes de identificacao pessoal, dados financeiros, autenticacao, comunicacoes pessoais, localizacao, historico de navegacao ou outros dados sensiveis.

Permissao de host e conteudo de pagina

A extensao usa content scripts para modificar o DOM da pagina localmente no navegador (por exemplo, substituicao de imagens e termos de texto). Esse processamento ocorre no dispositivo do usuario e nao e transmitido para servidores externos.

Codigo remoto

Nao usamos codigo remoto executavel. O codigo da extensao e empacotado junto ao item publicado.

Retencao e compartilhamento

Como nao coletamos dados de usuario, nao ha retencao nem compartilhamento de dados.

Alteracoes desta politica

Esta politica pode ser atualizada para refletir mudancas tecnicas ou regulatórias. Quando houver alteracao relevante, a pagina sera atualizada.
