saganIZER

Carl Sagan. Everywhere.

O saganIZER e uma extensao de navegador com um objetivo unico: aplicar um tema visual com referencias ao Carl Sagan nas paginas visitadas.

Repositorio no GitHub Politica de Privacidade

Unico proposito

Substituir elementos visuais e alguns termos de texto na pagina aberta para criar uma experiencia tematica. A extensao atua no conteudo exibido ao usuario e nao oferece funcionalidades paralelas.

Como funciona

Content script

Executa no contexto da pagina para ajustar o DOM localmente no navegador do usuario.

Sem backend proprio

O projeto nao possui API/servidor proprio para coletar ou armazenar dados de uso.

Escopo de host amplo

Necessario para aplicar o efeito de forma consistente nos sites visitados.
